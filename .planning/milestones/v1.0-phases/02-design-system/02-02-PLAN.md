<objective>
Integrate Coil 3.4.0 for all image and thumbnail loading, replacing the manual BitmapFactory + LruCache pattern in PagesAdapter and adding first-page thumbnail display in HistoryAdapter.

Purpose: The manual bitmap loading in PagesAdapter is technical debt that carries OOM risk, lifecycle leak risk (adapterScope never cancelled), and produces stale images on fast scroll because the .tag trick is incompatible with Coil. HistoryAdapter shows no thumbnail at all — a blank document list is a poor UX.
Output: Both adapters use Coil. Thumbnails have correct placeholders and crossfade. No manual coroutine scope remains in adapters.
</objective>
