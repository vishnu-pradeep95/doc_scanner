```kotlin
    fun applyDocumentBw(bitmap: Bitmap): Bitmap

}
```

- MAGIC and SHARPEN cap dimensions to maxDim=3368 when width or height exceeds that.
- DOCUMENT_BW creates a new bitmap (assertNotSame passes).

CRITICAL: enum value is DOCUMENT_BW, not BW or GRAYSCALE.
