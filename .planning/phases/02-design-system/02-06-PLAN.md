**Page indicator TextView (tvPageInfo) — convert to design-time placeholder:**

The TextView with `android:id="@+id/tvPageInfo"` currently has `android:text="1 / 1"`. This value is set programmatically by PdfEditorFragment (the text is updated dynamically as the user navigates pages). The hardcoded value is only a design-time preview.

Change:
```xml

```
To:
```xml

```

The `tools:` namespace is already declared at the root ConstraintLayout (`xmlns:tools="http://schemas.android.com/tools"`), so no new namespace declaration is needed. The tools:text attribute is stripped at build time and does not appear in the production APK — it only aids layout preview in Android Studio.
