Build the release APK and verify the automated aspects of the security audit:

1. Run `./gradlew assembleRelease` to build the release APK with R8 minification
2. Verify R8 ProGuard rules include log stripping: `grep -r "Log" app/proguard-rules.pro` should show rules stripping Log.v/d/i

5. Verify backup rules exclude encryption keyset: check `app/src/main/res/xml/backup_rules.xml` or data extraction rules
6. Verify FLAG_SECURE is set: `grep "FLAG_SECURE" app/src/main/java/com/pdfscanner/app/MainActivity.kt`
7. Verify EXTRA_IS_SENSITIVE is set: `grep "EXTRA_IS_SENSITIVE" app/src/main/java/com/pdfscanner/app/ui/PagesFragment.kt`
8. Verify accessibilityDataSensitive is set: `grep -r "accessibilityDataSensitive" app/src/main/res/layout/`
9. Verify RootDetector exists and is wired: `grep "RootDetector" app/src/main/java/com/pdfscanner/app/MainActivity.kt`
10. Verify InputValidator path validation: `grep "canonicalPath" app/src/main/java/com/pdfscanner/app/util/InputValidator.kt`
11. Verify SecureFileManager encryption: `grep "StreamingAead\|STREAMING_AEAD" app/src/main/java/com/pdfscanner/app/util/SecureFileManager.kt`
12. Verify SecurePreferences encryption: `grep "EncryptedSharedPreferences" app/src/main/java/com/pdfscanner/app/util/SecurePreferences.kt`
