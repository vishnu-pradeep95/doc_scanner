<objective>
Fix resource leaks in all PDF handling code, add stale temp file cleanup on app startup, and remove non-functional undo/redo buttons from PDF Editor.

Purpose: PDF operations (merge, split, compress, view, annotate) leak PdfRenderer and ParcelFileDescriptor resources on exception paths because they use manual close() instead of use {} blocks. Temp files from PDF viewing and editing accumulate in cacheDir with no cleanup. The undo/redo buttons in PDF Editor show "coming soon" toasts which is unacceptable in a portfolio app.

Output: All Closeable resources wrapped in use {} blocks, temp file cleanup logic in MainActivity.onCreate, and undo/redo buttons removed from PDF Editor layout and code.
</objective>

**MainActivity.kt -- BUG-03: Add temp file cleanup on startup:**

Add a `cleanupStaleTempFiles()` private function and call it from `onCreate()` after `setContentView()`:

```kotlin
override fun onCreate(savedInstanceState: Bundle?) {
    // ... existing theme/binding code ...
    setContentView(binding.root)

    // Clean up stale temp files from previous sessions
    cleanupStaleTempFiles()

    // ... existing navigation setup ...
}

private fun cleanupStaleTempFiles() {
    try {
        val cacheDir = cacheDir
        val oneHourAgo = System.currentTimeMillis() - (60 * 60 * 1000)

        cacheDir.listFiles()?.forEach { file ->
            if (file.isFile && file.lastModified() < oneHourAgo) {
                val name = file.name
                if (name.startsWith("pdf_view_temp_") ||
                    name.startsWith("temp_edit_") ||
                    name.startsWith("pdf_compress_temp")) {
                    file.delete()
                }
            }
        }

        // Also clean pdf_compress_temp directory
        val compressTemp = File(cacheDir, "pdf_compress_temp")
        if (compressTemp.exists() && compressTemp.isDirectory) {
            compressTemp.listFiles()?.forEach { file ->
                if (file.lastModified() < oneHourAgo) {
                    file.delete()
                }
            }
            if (compressTemp.listFiles()?.isEmpty() == true) {
                compressTemp.delete()
            }
        }
    } catch (e: Exception) {
        // Cleanup is best-effort, don't crash the app
        android.util.Log.w("MainActivity", "Temp cleanup failed", e)
    }
}
```
Add `import java.io.File` to MainActivity.kt if not already present.
